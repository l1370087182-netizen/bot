## 📈 交易策略详情

### 核心策略：动态双过滤器 + v10.0 三层防护

#### 1. 入场条件（信号生成）
```
时间周期: 30m (主要) + 1H/4H (宏观趋势确认)
指标组合:
  - ADX: 趋势强度 (要求 > 25 为强趋势)
  - StochRSI: 超买超卖 (0-20超卖, 80-100超买)
  - 布林带: 波动率过滤
  - MFI: 资金流验证

入场规则:
  LONG:  1H趋势UP + ADX>10 + StochRSI<45 + 价格<上轨
  SHORT: 1H趋势DOWN + ADX>10 + StochRSI>55 + 价格>下轨
```

#### 2. 信号质量评分 (SignalQualityScorer)
```
评分维度:
  - ADX强度 (30%): ADX>25得满分
  - 趋势一致性 (25%): 多时间周期趋势一致
  - 突破质量 (25%): 布林带突破有效性
  - 资金费率 (20%): 正费率利空多头，负费率利空空头

阈值:
  - PREMIUM (>=80): 1.5x仓位
  - STANDARD (70-80): 1.0x仓位
  - REDUCED (60-70): 0.5x仓位
  - REJECTED (<60): 不开仓
```

#### 3. 仓位管理 (PositionSizer + Pyramiding)
```
单笔风险: 账户余额2%
杠杆: 动态调整 (10x/7x/5x/3x/2x 基于ATR)

金字塔加仓 (盈利后):
  - Level 1: 盈利2%  → 加仓40%
  - Level 2: 盈利5%  → 加仓30%
  - Level 3: 盈利8%  → 加仓20%
  最大加仓: 3次，总仓位不超过账户6%
```

#### 4. 出场管理 (ExitManager)
```
分级止盈:
  - Stage 1: 盈利3% → 平仓20%
  - Stage 2: 盈利5% → 平仓30%
  - Stage 3: 剩余50% 移动止损 (2.8x ATR)

硬止损: 2% (固定)
移动止损: 基于15m ATR的2.8倍
```

#### 5. 账户保护 (AccountGuardian)
```
三层防护:
  - 日亏损 >=5%: 24小时暂停交易
  - 回撤 >=7%: 进入保命模式 (5x杠杆，半仓，禁止加仓)
  - 回撤 >=10%: 永久锁定，需手动解锁
```

#### 6. 币种分组 (CoinGrouper)
```
分组限制:
  - MEME组: 最多1个仓位 (DOGE, SHIB等)
  - PLATFORM组: 最多2个仓位
  - DEFI组: 最多2个仓位
  - LAYER1组: 最多3个仓位
防止相关性过高的币种集中持仓
```
