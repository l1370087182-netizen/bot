## 已知风险

- 当前系统仍是低到中等胜率的趋势型策略，不是高胜率均值回归策略
- 全样本仍强于样本外，说明依旧要警惕拟合
- 执行摩擦、滑点、funding 扰动仍会侵蚀边际，但这一版韧性已经明显提升
- `BNB` 虽然总收益略正，但质量一般，后续仍值得继续观察

## 下一步建议

1. 当前最合理的决策点是：先判断是否已经进入“小资金实盘观察”阶段，而不是默认继续榨干参数。
2. 如果继续优化，优先看 `BNB` 和 `TRX` 是否还要做方向分层。
3. 如果你已经对当前规则策略满意，可以开始小资金实盘观察。
4. 只有在当前规则策略稳定后，再引入 ML/AI 做增强层。

## ML/AI 可扩展方向

- 可以加入 ML/AI，但更适合先做过滤器和风险分层
- 当前阶段不建议让 ML 直接决定开平仓
