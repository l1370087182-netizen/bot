## 🏗️ 核心架构：三层防护 + 放大器

### 第一层：风险管理层 (Risk Layer)

#### 1. PositionSizer (仓位计算器)
- **功能**: 根据账户余额和风险百分比计算开仓数量
- **算法**: `position_size = (balance × risk_per_trade) / stop_distance`
- **默认**: 单笔风险 2% (可配置)
- **特点**: 自动考虑杠杆和最小名义价值限制

#### 2. AccountGuardian (账户守护者)
三层递进式保护：

| 层级 | 触发条件 | 保护措施 | 恢复方式 |
|------|----------|----------|----------|
| **日亏损保护** | 单日亏损 ≥5% | 暂停交易24小时 | 自动恢复 |
| **保命模式** | 回撤 ≥7% | 杠杆降至5x、仓位减半、禁止加仓 | 回撤恢复后自动解除 |
| **硬止损** | 回撤 ≥10% | 永久锁定，禁止所有交易 | 需手动解锁 |

#### 3. CoinGrouper (币种分组器)
- **目的**: 防止相关性过高的币种集中持仓
- **分组规则**:
  - MEME组 (DOGE/SHIB等): 最多1个仓位
  - PLATFORM组 (BNB/SOL等): 最多2个仓位
  - DEFI组 (UNI/AAVE等): 最多2个仓位
  - LAYER1组 (ETH/ADA等): 最多3个仓位

### 第二层：策略增强层 (Strategy Layer)

#### 4. SignalQualityScorer (信号质量评分器)
100分制多维度评分系统：

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| ADX强度 | 30% | >30得100分，>20得80分，>15得60分 |
| 趋势一致性 | 25% | 1H/4H同向100分，反向50分 |
| 突破质量 | 25% | 布林带挤压+放量100分 |
| 资金费率 | 20% | 方向有利100分，不利20分 |

**交易等级**:
- PREMIUM (≥75分): 1.5x仓位
- STANDARD (60-75分): 1.0x仓位
- REDUCED (50-60分): 0.5x仓位
- REJECTED (<50分): 不交易

#### 5. Dynamic Leverage (动态杠杆)
根据波动率自动调整杠杆倍数：

| ATR% | 杠杆 | 适用场景 |
|------|------|----------|
| <0.5% | 10x | 低波动，趋势明确 |
| 0.5-1% | 7x | 中等波动 |
| 1-2% | 5x | 较高波动 |
| 2-3% | 3x | 高波动 |
| >3% | 2x | 极高波动，风险大 |

### 第三层：盈利放大层 (Amplifier Layer)

#### 6. PyramidingManager (金字塔加仓器)
**只在盈利后加仓，不摊平亏损**：

| 盈利水平 | 加仓比例 | 累计仓位 |
|----------|----------|----------|
| 初始 | 100% | 100% |
| +2% | +40% | 140% |
| +5% | +30% | 170% |
| +8% | +20% | 190% |

**限制**: 总仓位不超过账户余额的6%

#### 7. ExitManager (出场管理器)
分级止盈 + 移动止损：

**分级止盈**:
- 盈利3% → 平仓20%
- 盈利5% → 平仓30%
- 剩余50% → 移动止损

**移动止损**: 基于15m ATR的2.8倍

**锁利机制**:
- 保本锁: 盈利1.8%后止损移至成本价+0.5%
- 5%锁利: 盈利5%后锁定至少3%利润

## 📊 入场策略 (v10.1 放宽版)

### 多头入场条件
```
✅ 1H趋势 = UP (价格 > EMA200)
✅ ADX > 8 (趋势强度足够)
✅ StochRSI < 50 (低位)
✅ StochRSI 金叉 (K线上穿D线)
✅ 非布林带挤压期
✅ 波动率合格 (0.6x < ATR < 3.5x)
✅ 信号评分 >= 60分
```

### 空头入场条件
```
✅ 1H趋势 = DOWN (价格 < EMA200)
✅ ADX > 8
✅ StochRSI > 50 (高位)
✅ StochRSI 死叉 (K线下穿D线)
✅ 非布林带挤压期
✅ 波动率合格
✅ 信号评分 >= 60分
```

### 与 v9.0 的差异
| 参数 | v9.0 (严格) | v10.1 (放宽) |
|------|-------------|--------------|
| ADX阈值 | >10 | >8 |
| StochRSI多头 | <45 | <50 |
| StochRSI空头 | >55 | >50 |
| MFI限制 | 有 | 移除 |
| 反弹检测 | 有 | 移除 |
| 信号评分门槛 | 70分 | 60分 |

## 🖥️ Web Dashboard 功能

### 1. 机器人控制面板
- **启动/停止/重启** 按钮
- **实时状态** 指示器 (运行/停止)
- **操作反馈** 消息提示

### 2. 实时监控
- **账户余额**: 实时USDT余额
- **今日盈亏**: 当日盈亏金额和百分比
- **胜率**: 30天胜率统计
- **净利润**: 30天净利润

### 3. 持仓详情
- **币种/方向/数量**: 基础信息
- **入场价/标记价**: 价格跟踪
- **盈亏/盈亏%**: 实时盈亏
- **杠杆/强平价**: 风险指标
- **等级**: PREMIUM/STANDARD/REDUCED

### 4. 风控状态
- **日亏损限制**: 5%进度条
- **回撤保护**: 7%保命模式状态
- **硬止损**: 10%锁定状态

### 5. 日志查看器
- **实时日志流**: 自动刷新
- **颜色区分**: 
  - 🟢 INFO: 普通信息
  - 🔵 TRADE: 交易信号
  - 🟡 WARNING: 警告
  - 🔴 ERROR: 错误
- **搜索功能**: 关键词搜索历史日志

## 💾 数据持久化

### SQLite 数据库 (trades.db)

**trades 表**:
```sql
- id: 交易ID
- symbol: 交易对
- side: 方向 (BUY/SELL)
- entry_price: 入场价
- exit_price: 出场价
- quantity: 数量
- pnl: 盈亏金额
- pnl_pct: 盈亏百分比
- entry_time: 入场时间
- exit_time: 出场时间
- signal_score: 信号评分
- max_profit_pct: 最大盈利%
- tags: 标签 (JSON)
```

**性能指标表**:
- 胜率、盈亏比、夏普比率
- 每日/每周/每月统计
